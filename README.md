# RDR2-base

Base repository scaffolding for RedM/RDR2 operational setup.

## Overview
This repo will host operational assets, scripts, and configuration for a RedM/RDR2 server environment. It currently initializes a clean structure to build upon.

## Structure
- `src/` — source configs, resources, and server assets
- `scripts/` — Windows PowerShell scripts for setup and maintenance
- `docs/` — documentation for operators and contributors

## Getting Started
1. Ensure you have Windows PowerShell 5.1 or later.
2. Explore `scripts/` for helper commands.
3. Add your server resources under `src/`.
