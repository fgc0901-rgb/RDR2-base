# Contribuindo

Obrigado por contribuir com o RDR2-base!

## Fluxo de Trabalho
- Crie branches de feature a partir de `main`.
- Abra PRs com descrição clara e notas de teste.
- Use commits convencionais quando possível.

## Padrões
- Mantenha mudanças focadas e mínimas.
- Atualize a documentação para qualquer mudança operacional.

## Setup Local
- Ambiente Windows com PowerShell 5.1.
- Scripts em `scripts/` fornecem tarefas comuns.
