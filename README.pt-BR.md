# RDR2-base

Base de repositório para configuração operacional do RedM/RDR2.

## Visão Geral
Este repositório abrigará ativos operacionais, scripts e configurações para um ambiente de servidor RedM/RDR2. No momento, ele inicializa uma estrutura limpa para evoluir.

## Estrutura
- `src/` — configurações, recursos e ativos do servidor
- `scripts/` — scripts do Windows PowerShell para setup e manutenção
- `docs/` — documentação para operadores e contribuidores

## Início Rápido
1. Garanta o Windows PowerShell 5.1 ou superior.
2. Explore `scripts/` para comandos auxiliares.
3. Adicione seus recursos de servidor em `src/`.
